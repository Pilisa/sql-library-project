# Library SQL Project

## 📌 Overview
This is a simple Library Management SQL project built for learning and QA practice. It demonstrates basic database design and SQL operations.

## 📂 Tables
- Books
- Members
- BorrowedBooks

## 🧠 Skills Demonstrated
- SQL Table Creation (DDL)
- Data Insertion (DML)
- SELECT Queries
- JOIN operations

## 🚀 How to Use
Run `schema.sql` first, then `data.sql`, then `queries.sql`.
